# paradigm architecture

Paradigm is a local-first, audio-only AI companion designed for grounding, reflection, and clarity.

## Core components

- audio pipeline  
- speech-to-text (local)  
- companion core  
- text generation (local LLM)  
- text-to-speech (optional)  
- safety + behavior contract  
- CLI interface  
- configuration system  

## Flow

audio → text → grounding response → output
