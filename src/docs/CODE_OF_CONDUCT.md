# Code of Conduct for paradigm

Paradigm is a project built on clarity, autonomy, privacy, and respect.  
This Code of Conduct exists to ensure that everyone who engages with the project does so in a way that supports those values.

This project is not just software — it is an ethical commitment.

---

## 1. Our Standards

Contributors and community members are expected to:

- Treat others with respect and kindness  
- Communicate clearly and constructively  
- Honor the project’s principles of privacy and autonomy  
- Avoid harmful, manipulative, or coercive behavior  
- Support a safe, inclusive, and thoughtful environment  

Unacceptable behavior includes:

- Harassment, discrimination, or personal attacks  
- Dismissive or hostile communication  
- Attempts to push the project toward unsafe or unethical directions  
- Introducing features that violate privacy or autonomy  
- Encouraging emotional dependency on AI systems  

---

## 2. Project Values

Paradigm is grounded in:

- **Local-first design**  
- **Privacy and data minimization**  
- **Emotional safety**  
- **Transparency**  
- **User autonomy**  
- **Non-replacement of human connection**  

Any contribution that conflicts with these values will not be accepted.

---

## 3. Scope

This Code of Conduct applies to:

- GitHub issues  
- Pull requests  
- Discussions  
- Documentation  
- Any community spaces related to the project  

---

## 4. Enforcement

Project maintainers may take action in response to unacceptable behavior, including:

- Warning the contributor  
- Closing issues or pull requests  
- Removing contributions that violate safety or ethics  
- Blocking individuals from participating  

Enforcement decisions are made to protect the integrity and safety of the project.

---

## 5. Reporting

If you encounter behavior that violates this Code of Conduct, please open an issue or contact the maintainers directly.

Reports will be handled respectfully and privately.

---

## 6. Commitment

By participating in this project, you agree to uphold this Code of Conduct and contribute to a community built on clarity, respect, and autonomy.

Paradigm exists to support people — not to control them.  
Our community must reflect that.
