# Contributing to paradigm

Thank you for your interest in contributing to **paradigm**, a local-first, audio-only AI companion designed to provide grounding, reflection, and clarity.

This project is built on strong principles:
- user autonomy  
- privacy  
- offline functionality  
- ethical boundaries  
- transparency  
- simplicity  

Contributions are welcome as long as they align with these values.

---

## 1. Project Philosophy

Paradigm is not a chatbot, not a therapist, and not a replacement for human connection.

Contributions must respect the core principles:

- **Local-first:** No cloud dependencies.  
- **Privacy:** No logs, no recordings, no hidden data collection.  
- **Safety:** No medical advice, no diagnosis, no emotional dependency.  
- **Clarity:** The companion reflects, grounds, and supports — it does not control.  
- **Simplicity:** Small, intentional, minimal.  

If a feature conflicts with these principles, it will not be accepted.

---
